Windows 10 / 11 (x64) ・ Android 8.0 以降

特長

🔌

USB のみで完結

ADB 経由で映像とタッチイベントを伝送。Wi-Fi 接続や HDMI キャプチャ機器、 外付けディスプレイは不要です。

🖥️

本物の拡張ディスプレイ

仮想ディスプレイドライバー (MttVDD) を同梱。Windows からは 通常のセカンドモニターとして認識され、ウィンドウのドラッグ移動で タブレットへ自然に拡張できます。

⚡

低遅延 H.264

ハードウェアエンコーダー (Media Foundation) で 60fps を実現。 DXGI Desktop Duplication でフレームを直接 GPU から取得します。

🔐

UAC ダイアログなし

インストール時に SYSTEM 権限の Scheduled Task を登録するため、 アプリ使用中はユーザーアカウント制御の確認が一切表示されません。

🎁

Windows 版は無料

Windows 用クライアントは完全無料で利用可能。Android 側は購入前に 3 回まで無料でお試しいただけます。

動作イメージ

ノートパソコン本体のディスプレイ (#1) と、USB 接続した Android タブレット (#2) が拡張デスクトップとして連携します。

動作環境

Windows (送信側)

  • Windows 10 1809 / Windows 11 (x64)
  • .NET 8 Desktop Runtime（インストーラーに同梱）
  • Windows App Runtime 2.1（同梱）
  • USB ポート (Type-A もしくは Type-C)

Android (受信側)

  • Android 8.0 (API 26) 以降
  • USB デバッグ ON
  • 初回 3 回まで無料 / 以降は買い切り
